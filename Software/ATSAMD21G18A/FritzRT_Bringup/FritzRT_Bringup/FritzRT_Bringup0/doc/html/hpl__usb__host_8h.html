◆ usb_h_pipe_cb_xfer_t

typedef void(* usb_h_pipe_cb_xfer_t) (struct usb_h_pipe *pipe)

Prototyping USB HCD callback of pipe transfer done. For control pipe, it's forced to call even if there is no transfer in progress, since the pipe size could be changed at run time.
