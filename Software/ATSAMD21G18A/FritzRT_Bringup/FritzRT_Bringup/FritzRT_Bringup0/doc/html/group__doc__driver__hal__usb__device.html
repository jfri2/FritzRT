◆ usb_d_ep_cb_more_t

typedef bool(* usb_d_ep_cb_more_t) (const uint8_t ep, const uint32_t count)

Callback that is invoked when buffer is done without error, but last packet is full size packet without ZLP. Return true if more data has been requested.

◆ usb_d_ep_cb_setup_t

typedef bool(* usb_d_ep_cb_setup_t) (const uint8_t ep, const uint8_t *req)

Callback that is invoked when setup packet is received. Return true if request has been handled, or control endpoint will stall IN/OUT transactions.

◆ usb_d_ep_cb_xfer_t

typedef bool(* usb_d_ep_cb_xfer_t) (const uint8_t ep, const enum usb_xfer_code code, void *param)

Callback that is invoked when all data is finished, including background transfer, or error happens. In control transfer data stage, return value is checked, return false if no error happens.

◆ usb_d_ep_transfer()

int32_t usb_d_ep_transfer ( const struct usb_d_transfer *  xfer)

Endpoint transfer.

For control endpoints, start the transfer according to the direction in the bmRequest type, and finish with STATUS stage. For non-control endpoints, the transfer will be unique direction. Defined by bit 8 of the endpoint address.

Parameters
