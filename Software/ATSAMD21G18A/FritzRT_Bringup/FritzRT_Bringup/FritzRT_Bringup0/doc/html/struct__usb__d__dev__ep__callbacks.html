_usb_d_dev_ep_cb_more_t _usb_d_dev_ep_callbacks::more

Callback that is invoked to check if buffer is NULL and more data is required. It's called when last packet is full size packet, without auto ZLP enabled. It could be called when background transfer is still in progress.
